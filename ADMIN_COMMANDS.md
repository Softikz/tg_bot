# 🔐 Команды администратора

## Настройка

Перед использованием административных команд необходимо установить ID вашего аккаунта в файле `config.py`:

```python
ADMIN_ID = 5748972158
```

### Как узнать свой user_id?

1. Отправьте боту команду `/start`
2. Посмотрите в логи консоли (там будет выведен ваш ID при первом сообщении)
3. Или используйте другого бота, который показывает user_id (например, @userinfobot в Telegram)

---

## Команды

### `/broadcast` - Рассылка сообщений всем пользователям

**Описание**: Отправляет сообщение всем зарегистрированным пользователям бота.

**Использование**:
1. Отправьте команду `/broadcast`
2. Введите сообщение, которое нужно разослать
3. Бот автоматически отправит сообщение всем пользователям

**Пример**:
```
/broadcast
[Введите сообщение]
```

После отправки вы получите статистику:
- 📤 Отправлено (количество успешно доставленных сообщений)
- ❌ Ошибок (количество пользователей, которым не удалось отправить)

**Отмена**: Нажмите кнопку "❌ Отмена" или используйте `/cancel`

---

## Советы

- ✅ Сообщение поддерживает HTML-форматирование (жирный текст, курсив, ссылки)
- ⏱️ Рассылка может занять время если пользователей много
- 📊 Проверяйте логи для диагностики проблем с доставкой
- 🔒 Никому не давайте свой user_id для избежания несанкционированного доступа

---

## Технические детали

Функция рассылки:
- Перебирает всех пользователей из базы данных `users.json`
- Отправляет каждому текстовое сообщение с объявлением
- Пропускает пользователей, заблокировавших бота или удаливших чат
- Логирует статистику успешных и неудачных доставок
